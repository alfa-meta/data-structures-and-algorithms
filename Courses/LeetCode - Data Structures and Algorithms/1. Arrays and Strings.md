Mutable - a type of data that can be changed
Immutable - a type of data that cannot be changed

Changing Immutable data requires recreating it.

## Two Pointers

Two pointers - involves having two integer variables that both move along an iterable.
	O(n) - runtime in the loop

```Basic example of two pointers
1. Start one pointer at the first index '0' and the other pointer in the last index 'input.length - 1'

2. Use a while loop until hte pointers are equal to each other

3. At each iteration of the loop, move the pointers towards each other. This means either increment the pointer that started at the first index, decrement the pointer that started at the last index, or both.
```

### Example Answer 1
```Python3
def check_for_target(nums, target):
    left = 0
    right = len(nums) - 1

    while left < right:
        # curr is the current sum
        curr = nums[left] + nums[right]
        if curr == target:
            return True
        if curr > target:
            right -= 1
        else:
            left += 1
    
    return False
```

def combine(arr1, arr2):
    # ans is the answer
    ans = []
    i = j = 0
    while i < len(arr1) and j < len(arr2):
        if arr1[i] < arr2[j]:
            ans.append(arr1[i])
            i += 1
        else:
            ans.append(arr2[j])
            j += 1
    
    while i < len(arr1):
        ans.append(arr1[i])
        i += 1
    
    while j < len(arr2):
        ans.append(arr2[j])
        j += 1
    
    return ans
