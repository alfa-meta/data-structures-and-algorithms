# Notes
In this book the term `client` refers to a program that calls a method in another library and the term implementation to describe the Java code that implements the methods in an API.



Primitive Data types - a data type is a set of values and a set of operations on those values.

Primitives are language specific, Python primitives include:
- `int` - whole numbers `42`, `-7`
- `float` - decimal numbers `3.14`, `-0.5`
- `bool` - True or False (technically a subclass of int)
- `str` - `"hello"`
- `None` - `NoneType` or Absence of value


Variable is named with an identifier.
Each variable has an associated data type value.


Expression - is a literal followed by an operator and a literal again.
	This pattern of `L` `O` `L`  can be chained such as L + L * L


## Arrays

Array - stores a sequence of values that are all of the same type.
	From 0 to N - 1 number of elements.
	Where a[i] represents the `ith` element in the array.
	N = number of elements.


Two-Dimensional Array - M by N where M being rows of length N.


## Recursion

- Must have a base case - always include a conditional statement as the first statement in the program that has a return.
- Recursive calls must address sub-problems that are smaller in some sense, so that recursive calls converge to the base case.
- Recursive calls should not address sub-problems that overlap.
